How to Run?

README

본 서비스를 100% 믿지는 마십시오.

본 서비스는 파싱을 해서 푸시를 알려주는 구조로 만들어졌습니다. 따라서 100% 신뢰하여 사용하기에는 다소 무리가 있습니다. 본 서비스는 언제든지 서비스를 중단할 수 있으며, 각 교수님별 급작스런 서비스 중단이나 일시적인 멈춤이 있을 수 있습니다. 물론 서비스가 멈출때에도 사용자에게 알려주는 기능을 탑재할 것이지만, 이러한 노력에도 불구하고 저희 개발자는 이 서비스를 100% 신뢰하지는 말아달라고 말씀드리고 싶습니다.

그렇다해서 저희들은 잘못된 정보를 제공하는것을 방치해두지는 않습니다. 최대한 정확한 정보를 제공하고 빠른 정보를 제공하고자 함이 저희 서비스의 목표입니다!

버그리포팅과 피드백은 언제나 환영이며, 안드로이드 개발자(김동민), 웹 개발자(이승민), 서버 개발자(곽도영)에게 말씀해주시면 빠른시일내에 버그를 해결하고 회의를 통해 피드백을 반영시키겠습니다.
